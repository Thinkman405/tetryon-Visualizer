# GHOST TERMINAL

Visual Interface for Project Lynchpin.

## Setup

This project uses Vite + React.

1. `npm install`
2. `npm run dev`

## Architecture

- **lib/TetryonGeometry.js**: Core physics engine.
- **lib/TCIEngine.js**: State management and mesh topology.
- **components/TetryonMap.jsx**: Visualization layer.

## Aesthetics
Dark mode only. Tetryon Neon accents.
